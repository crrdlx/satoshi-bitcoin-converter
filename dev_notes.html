Satoshi Bitcoin Converter - Dev Notes


Howdy. Some notes from the Satoshi Bitcoin Converter developer:

Dogecoin notes:

  • I'm aware that the Dogecoin converter has issues. Specifically, because DOGE is currently around the $0.05 USD mark, the program rounds up or down to either 0.01 or 0.00. I realize that this is not ideal.
  • I plan to investigate this issue, and will try to fix it if I can. It seems like it should be easy enough, but coding can be finicky. There are no guarantees.
  • I included DOGE because I love it and the DOGE community. It's there out of appreciation and for the fun of it.
Notes in general:
